#  Word Counter App

A simple web application that counts the number of **words** and **characters** in your text.

##  Features

- Count total **words**
- Count total **characters**
- Instant result as you type
- Clean and minimal UI

##  Technologies Used

- HTML
- CSS
- JavaScript

##  How to Use

1. Type or paste any text in the textbox.
2. The app will automatically display the word and character count.
